% rm1masm(1) v{{version}}

# NAME

rm1masm - MICRO-1 micro assembler written in Rust

# SYNOPSIS

rm1masm \[OPTIONS\] \<input\>

# OPTIONS

`-h` `--help`
: Print help

`-v` `--version`
: Print version

`-o` `--output`
: Set output path
